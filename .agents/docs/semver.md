# SemVer
Para o controle de numeração das versões do GSK é utilizado o versionamento semântico.

## Exemplo:
`2.1.0`

2 = MAJOR - alterações incompatíveis com a versão atual.
1 = MINOR - novas funcionalidades compatíveis com a versão MAJOR atual.
0 = PATCH - correções compatíveis com a versão MINOR atual.
