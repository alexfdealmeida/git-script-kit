# system

Esta pasta contém scripts prontos para uso em produção.

## Objetivo 
Armazenar scripts, funções e constantes utilizados sistematicamente pelos demais scripts do GSK.

## Regras
- Acesso permitido para os perfis `[dev]` e `[scm]`.

## Observação
Por se tratar de scripts amplamente utilizados no GSK, possíveis alterações devem ser analisadas de forma detalhada, para evitar impactos na performance dos demais scripts.
