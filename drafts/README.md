# drafts

Esta pasta contém scripts em desenvolvimento, ainda NÃO validados para uso em produção.

## Objetivo 
Armazenar scripts que ainda estão em fase de testes, ajustes ou validação. Ou seja, podem estar incompletos ou não seguir ainda o padrão definitivo do projeto.

## Regras
- Acesso permitido APENAS para o perfil `[scm]`.
- Antes de mover qualquer scripts para produção, valide seu funcionamento e revise o padrão de nomenclatura, mensagens e compatibilidade com os demais scripts do GSK.

## Observação
Os scripts NÃO são considerados estáveis e podem ser reescritos ou removidos sem aviso prévio.
